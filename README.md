# bychudy — портфолио (реплика)

Копия сайта-портфолио **[bychudy.com](https://www.bychudy.com)** «один в один»: метафора рабочего
стола macOS — выцветающий портрет-обои, разбросанные «иконки» проектов с подписями и док снизу.
**Клик по иконке** открывает окно в стиле «Свойства файла» (Information about…) с описанием
проекта, типом работы и галереей превью. Чистый HTML/CSS/JS, без сборщиков и зависимостей.

```
index.html        — разметка (сцена, слои, окно «Get Info»)
styles.css        — стили: SF Pro, подписи, стеклянный док, окно, масштабирование, мобильная раскладка
script.js         — данные проектов/дока + рендер + подгонка сцены + открытие/закрытие/перетаскивание окна
windows-data.js   — контент окон (заголовок, подзаголовок, описание, Type, превью) для всех 27 проектов
assets/
  bg.png          — портрет-обои
  favicon.png
  icons/          — 27 превью проектов (иконки на «рабочем столе»)
  dock/           — 9 иконок дока (Ae, Ps, Ai, ⚠, Notes, Photos, Instagram, Mail, Trash)
  previews/       — изображения галереи внутри окон, по папке на проект
```

## Запуск

**Самый простой способ — двойной клик по `index.html`** (открывается в браузере, работает офлайн,
проверено в Safari и Chrome).

Если браузер по какой-то причине не подхватывает ассеты, поднимите локальный сервер:

```bash
cd <папка проекта>
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Как сделать своим

Контент правится в двух местах:

- **`script.js` → массив `PROJECTS`** — иконки на рабочем столе. У каждого: `slug` (имя файла
  в `assets/icons/`), `ext`, `label` (подпись, `<br>` = перенос строки), размер `w`×`h` и координаты
  центра/верха для десктопа (`dcx`,`dy`, холст **1440×900**) и мобайла (`mcx`,`my`, холст **390×844**).
- **`script.js` → массив `DOCK`** — иконки дока по порядку. `sep:true` = разделитель,
  `href` превращает иконку в ссылку (сейчас Instagram и e-mail).
- **`windows-data.js` → объект `WINDOWS`** — содержимое окна каждого проекта (по `slug`):
  `title`, `big`, `subtitle`, `desc`, `type`, `previews` (массив `{src,w,h}`).

Чтобы подставить свои работы: замените картинки в `assets/icons/` и `assets/previews/<slug>/`,
обои `assets/bg.png`, поправьте подписи/координаты в `PROJECTS` и тексты в `WINDOWS`,
ссылки Instagram/почты — в `DOCK`.

## Опубликовать в интернете (бесплатно)

Сайт статический, поэтому подойдёт любой статический хостинг — собирать ничего не нужно,
просто загрузите всю папку:

- **Netlify Drop** — netlify.com/drop, перетащить папку в окно браузера.
- **Vercel** — `npx vercel` в папке проекта.
- **GitHub Pages** — закоммитить в репозиторий и включить Pages.
- **Cloudflare Pages** — подключить репозиторий или загрузить папку.

## Шрифт

Подписи используют системный **SF Pro** через `-apple-system` (нативно на macOS/iOS).
На других ОС подхватится ближайший системный гротеск — при необходимости подключите
веб-шрифт через `@font-face` в `styles.css`.

## ⚠️ Важно про материалы

Портрет, обложки и превью проектов, тексты описаний, имя «bychudy», Instagram и e-mail
взяты с оригинального сайта и принадлежат его автору. Это точная визуальная реплика как
шаблон/референс — **перед публикацией замените изображения, тексты, имя и контакты на свои.**
